Online Documentation Instructions

To use an online copy of a library's documentation, simply have the url point to the root of the version of the documentation that you want to use. ex: https://docs.python.org/3/ or https://flogin.readthedocs.io/en/latest/

Offline Documentation Instructions

To use an offline copy of a library's documentation, generate the html version of the documentation (see the sphinx docs for help), and provide the path to the generated folder. For example, if I build the html docs from C:\Users\default.MyPC\AppData\Roaming\FlowLauncher\Plugins\flogin\docs, the last line that sphinx prints will look something like: The HTML pages are in _build\html. After concatenating those paths, this is the location of my documentation: C:\Users\default.MyPC\AppData\Roaming\FlowLauncher\Plugins\flogin\docs\_build\html
